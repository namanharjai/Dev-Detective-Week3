# Prompts Used During Development

These are the kinds of doubts and help prompts I used while building the project.

1. Explain `fetch()` and `async/await` in simple words with one GitHub API example.
2. Why do we use `response.ok` before converting the response to JSON?
3. How do I show a loading message while API data is being fetched in vanilla JavaScript?
4. How can I handle a 404 error from the GitHub API without breaking the UI?
5. How do I format a GitHub date like `2023-01-25T12:00:00Z` into `25 Jan 2023` in JavaScript?
6. How can I fetch a user's latest 5 repositories after getting the profile data?
7. How do I open a repository link in a new tab safely using `target="_blank"`?
8. Explain `Promise.all()` with a simple example using two API requests.
9. How can I compare two GitHub users side by side in JavaScript?
10. How do I calculate total stars by looping through a user's repositories?
11. Check my error messages and tell me if they sound clear and natural for users.
12. Review my JavaScript logic and explain the parts that can fail during API requests.
